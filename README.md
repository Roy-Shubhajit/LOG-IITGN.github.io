# Learning on Graphs Conference 2026 - Local Meetup Website

This is a GitHub Pages website for the Learning on Graphs Conference 2026 local meetup at IIT Gandhinagar.

## Setup Instructions

1. Create a new repository on GitHub named `<your-username>.github.io`
2. Clone the repository to your local machine
3. Extract the contents of this zip file into the repository directory
4. Push the files to your repository:
   ```
   git add .
   git commit -m "Add LoG 2026 meetup website"
   git push
   ```
5. Your website will be available at `https://<your-username>.github.io`

## File Structure

- `index.html` - Main website page
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality (navigation, tabs, scroll effects)

## Updating Content

To update content, edit the sections in `index.html`:

- **Keynote Speakers**: Update speaker cards in the speakers section
- **Schedule**: Modify the schedule tables for Day 1 and Day 2
- **Organizers**: Add more organizers by duplicating the organizer-card div
- **Dates**: Update important dates in the dates-grid section
- **Sponsors**: Update the sponsors section

## Features

- Responsive design (works on desktop, tablet, and mobile)
- Sticky navigation bar with smooth scrolling
- Tab switching between Day 1 and Day 2 schedules
- Professional color scheme
- Mobile hamburger menu
- Scroll-to-top button
- Hover animations and effects

## Colors Used
